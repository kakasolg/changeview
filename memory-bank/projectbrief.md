# 지혜의 렌즈 - Project Brief (2025-06-29 현황 반영)

## 📋 프로젝트 개요

**프로젝트명**: 지혜의 렌즈 (Wisdom Lenses)
**부제**: 64가지 관점으로 보는 세상의 모든 문제
**현재 상태**: 플래시카드 학습 시스템 (운영 중) + 8면체 주사위 시스템 (기술적 문제로 95% 완성 후 대기)

## 🎯 핵심 목표

### 1. **혁신적 괘 선택 시스템 (최종 목표)**
   - **8면체 주사위 방식**: 수학적으로 완벽한 64괘 조합 (8×8=64)
   - **능동적 참여**: 사용자가 직접 운명을 선택하는 주체적 경험
   - **주역 원리 부합**: 우연의 필연성, 동기감응 구현

### 2. **과학적 정신 모델 기반 해석**
   - **찰리 멍거 격자틀 이론**: 점술이 아닌 학문적 접근
   - **투자/경영 특화**: 실무에서 바로 활용 가능한 의사결정 도구
   - **다학문적 통찰**: 물리학, 경제학, 심리학 기반 분석

### 3. **실용적 학습 시스템 (현재 운영)**
   - **스페이스 반복 학습법**: 과학적 기억 정착 방식
   - **개인화 추적**: 개별 학습 패턴 및 진도 관리
   - **메모 시스템**: 사용자별 통찰 기록 및 누적

## 🎲 프로젝트 발전 단계

### Phase 1: 이상적 비전 설계 ✅ (완료)
- **8면체 주사위 64괘 시스템** 설계

- **찰리 멍거 정신 모델** 통합 계획

- **MongoDB + Gemini API** 연동 성공
- **팔괘 상징 표시** 기술적 문제 발생

### Phase 3: 실용적 대안 구현 ✅ (현재 운영)
- **플래시카드 학습 시스템** 완성

## 🃏 현재 운영 중인 시스템 특징

### 핵심 가치 제안
- **과학적 접근**: 찰리 멍거 정신 모델 기반 신뢰성
- **학습 효과**: 스페이스 반복으로 장기 기억 정착
- **개인화**: 사용자별 학습 패턴 추적 및 맞춤형 복습
- **실용성**: 복잡한 철학이 아닌 구체적이고 실행 가능한 통찰

### 사용자 경험
- **직관적 조작**: 카드 뒤집기로 간단한 학습
- **즉시 피드백**: 4단계 난이도 평가 (again/soon/later/mastered)
- **학습 동기**: 게임화된 진도 관리 시스템
- **지속적 성장**: 개인 메모 누적으로 통찰 발전

### 기술적 완성도
- **100% 동작**: 모든 기능 완전 구현 및 테스트 완료
- **안정성**: MongoDB 기반 신뢰성 있는 데이터 관리
- **확장성**: 새로운 기능 추가 용이한 구조
- **성능**: 빠른 응답 속도 및 원활한 사용자 경험

## 🚀 미래 발전 방향

### 단기 목표: 기술적 문제 해결
**목표**: 8면체 주사위 팔괘 상징 표시 완성
**방법**: UV 매핑, 3D 텍스트, 대안 라이브러리 검토
**결과**: 혁신적 3D 인터랙션 UX 실현

### 중기 목표: 다중 모드 통합
**플래시카드 모드**: 체계적 학습용 (현재 운영)
**주사위 모드**: 직관적 괘 선택용 (기술 해결 후)
**AI 통찰 모드**: 텍스트 기반 분석용 (백업 완료)

### 장기 비전: 완전한 의사결정 도구
- **투자 전문**: 리스크 관리 및 기회 포착 특화
- **경영 전략**: 조직 문제 해결 및 전략 수립
- **개인 성장**: 인생 선택 및 방향성 탐색
- **교육 플랫폼**: 동서양 융합 사고 교육

## 👥 타겟 사용자 (재정의)

### 주요 사용자층
1. **학습자** (현재 메인)
   - 64괘와 정신 모델을 체계적으로 학습하고 싶은 사람
   - 스페이스 반복으로 장기 기억에 정착시키려는 사람
   - 동양 철학과 서양 과학의 융합에 관심 있는 사람

2. **의사결정자** (미래 확장)
   - 투자, 경영, 전략 결정에 다각적 관점이 필요한 전문가
   - 찰리 멍거 스타일 격자틀 사고를 실무에 적용하려는 사람
   - 직관과 논리를 균형 있게 활용하려는 리더

3. **탐구자** (고급 사용자)
   - 주역의 원리를 현대적으로 해석하고 싶은 연구자
   - AI 기반 통찰과 전통 지혜의 접점을 찾는 사람
   - 혁신적 사고 도구에 관심 있는 얼리어답터

2. **신비성**: 고대 지혜의 깊이와 현대 기술의 만남
3. **직관성**: 복잡한 설명 없이 바로 이해 가능한 인터페이스
4. **신뢰성**: 과학적 근거와 논리적 해석 제공
